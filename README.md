# Customer Segmentation Using Machine Learning

This project uses customer data and clustering techniques to identify distinct customer groups based on their purchasing behavior.

## Project Overview

Customer segmentation helps businesses understand their customers and create targeted marketing strategies.

This project analyzes customer data and groups customers into different segments using machine learning algorithms.

## Features

* Data preprocessing
* Customer behavior analysis
* Clustering and segmentation
* Data visualization
* Business insights generation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

* Mall_Customers.csv

## Project Structure

* main.py
* Mall_Customers.csv
* Customer_Segments.csv

## How to Run

```bash
pip install pandas numpy matplotlib scikit-learn
python main.py
```
